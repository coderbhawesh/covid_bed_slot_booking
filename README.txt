1) Download Xammp from --> (https://www.apachefriends.org/download.html) and install it on the system.
2) Insert your own mysql password in line number 41 of main.py file.
3) Download all python dependency libraries shown in the requirements.txt file. 
(Go to command prompt and type command pip install -r requirements.txt)
4) Open phpMyAdmin from the Xampp control panel and create database as "coviddbms" and import the covid.sql file.

The installation part is done.

Now to run the website first we need to do admin login for the login credentials make sure to use the same as mentioned in the config.json file. Now you can add Hospital user. Also give a valid email id while making a user.
